ProtégéVOWL: VOWL Plugin for Protégé

VOWL Specification »

ProtégéVOWL is a Protégé plugin for the visualization of ontologies. It implements the Visual Notation for OWL Ontologies (VOWL) which provides graphical depictions for elements of the Web Ontology Language (OWL) that are combined to a force-directed graph layout representing the ontology.

Installation of ProtégéVOWL

  1. Copy ProtégéVOWL to the plugins folder of Protégé
  2. plugin folder of the Protégé installation.
  3. Start Protégé and Activate the ProtégéVOWL plugin
  4. Verify Installation
  5. Customizing ProtégéVOWL workspace tab

1. Copy ProtégéVOWL to the plugins folder of Protégé

ProtégéVOWL has been developed for ontology editor Protégé that needs to be downloaded and installed first. Can be used with Protégé version 4.x (or higher) ProtégéVOWL requires at least version 4.3 of Protégé Desktop that is available at protege.stanford.edu free of charge.

ProtégéVOWL consists of a JAR file that can be downloaded below. Download and copy the Protégé inside the <installation_path>\Protege_4.3\plugins\ folder. A Download link is provided below.

Download ProtégéVOWL plugin (beta 0.1) ↓

2. Start Protégé and Activate the ProtégéVOWL plugin

After the first step is completed, we are required to activate the plugin inside Protégé in order for it to be used.

Thus we need to start Protégé and from the Window menu (Window >> Tabs >> VOWL) we will activate the ProtégéVOWL plugin (see Figure 1 for details).

A new tab entitled VOWL will be created alongside existing Protégé tabs.

3. Verify Installation

After these two steps are finalized we will proceed to load a ontology within Protégé in oder to make use of the VOWL workspace tab and verify the installation.

VOWL workspace tab is structured as follows (see Figure 2 for details):

  • On the left side (canvas area) the loaded ontology can be visualized.
  • On the ride side (menu panel) additional information can be displayed when selecting an element from the visualization.
  • The bottom on the right side of the menu panel, several controls are available in oder to change the ontology visualization options.

4. Customizing ProtégéVOWL workspace tab

A workspace tab can be customized by utilizing the features offered by ProtégéVOWL under the Window menu (Window >> Views >> Ontology views >> VOWL Controls/VOWL Sidebar/VOWL Viewer) -- see Figure 3 for details.
